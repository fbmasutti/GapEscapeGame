# Escapa Buraco Arcade

Um projeto open-source inspirado nos clássicos jogos eletromecânicos de habilidade como *Ice Cold Beer* e *Zeke's Peak*. O objetivo é construir uma máquina arcade em MDF com controle por joysticks, utilizando eletrônica baseada em Arduino e componentes maker acessíveis. :contentReference[oaicite:0]{index=0}

## O projeto

O Escapa Buraco Arcade foi desenvolvido pelo **Coletivo Máquina Tudo** como uma plataforma aberta para estudos, fabricação digital e experimentação em eletrônica, marcenaria e programação. A estrutura foi projetada para corte a laser em MDF, com alguns acabamentos manuais na versão 1.0. :contentReference[oaicite:1]{index=1} :contentReference[oaicite:2]{index=2}

## Conteúdo do repositório
